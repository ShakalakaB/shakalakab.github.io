Remove Duplicate Value from Nested Associative Array: Part 1

Posted on 2020-03-22

Romoving duplicate value from nested array and return the result as json is kind of tricky, the main problem is
how to loop through and remove element while keeping array reindexed. I would compare three solutions I came up
and point out some thing we don’t notice usually.

Question

Supposing we have an array as following, and need to remove element which has duplicate ‘value’ in ‘channels’. So we need
to remove $arr[1]['channels'][0] and $arr[2]['channels'][0].

$unique = [];
foreach ($arr as &$item) {
foreach ($item[‘channels’] as $key => $channelItem) {
if (in_array($channelItem[‘value’], $unique)) {
unset($item[‘channels’][$key]);
continue;
}
$unique[] = $channelItem[‘value’];
}

print_r($arr);

$unique = [];
foreach ($arr as &$item) {
foreach ($item[‘channels’] as $key => $channelItem) {
if (in_array($channelItem[‘value’], $unique)) {
array_splice($item[‘channels’], $key, 1);
continue;
}
$unique[] = $channelItem[‘value’];
}

$unique = [];
foreach ($arr as &$item) {
array_walk($item[‘channels’], function ($channelItem, $key) use (&$unique, &$item) {
if (in_array($channelItem[‘value’], $unique)) {
array_splice($item[‘channels’], $key, 1);
}
$unique[] = $channelItem[‘value’];
});
} print_r($arr);
