Remove Duplicate Value from Nested Associative Array: Part 1

Posted on 2020-03-22 Edited on 2020-03-24

Romoving duplicate value from nested array and return the result as json is kind of tricky, the main problem is
how to loop through and remove element while keeping array reindexed. I would compare three solutions I came up
and point out some thing we don’t notice usually.

Question

Supposing we have an array as following, and need to remove element which has duplicate ‘value’ in ‘channels’. So we need
to remove $arr[1]['channels'][0] and $arr[2]['channels'][0].

*Notice: PHP-version 7.3.14

Solution 1: using foreach and unset

Result

This solution looks fine, but it would cause problem when sent through json response. Noticing the key in
$arr[1]['channels] and $arr[2]['channels], they are not in sequential, this could make the array after sent
through json difficult to loop.

Solution 2: using foreach and array_splice

array_splice would reindex array automatically after remove element.

Result

This solution haven’t filtered out duplicate value in$arr[2]['channels'], and even remove $arr[2]['channels'][3]
which should be kept. Why?
It’s because the internal array pointer in foreach is not synchronous with array_splice. Not matter how
array_splice reindex array, foreach would just keep the index as beginning, so same index in these two ways
would point to different value once reindex happens.
